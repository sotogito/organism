- 특수문자가 존재하는지

```java
private void validateNameSpecialChar(String name){  
    Matcher matcher = Pattern.compile("[!@#$%^&*().?\":{}|<>]").matcher(name);  
    if (matcher.find()) {  
        throw new IllegalArgumentException("특수문자는 사용하지 못합니다.");  
    }  
}
```
