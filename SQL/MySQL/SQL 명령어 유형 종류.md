### 1. DQL (Data Query Language) : 데이터 질의 언어
- SELECT : 데이터 조회
        
### 2. DML (Data Manipulation Language) : 데이터 조작 언어 
- INSERT : 데이터 삽입
- UPDATE : 데이터 수정
- DELETE : 데이터 삭제
### 3. DDL (Data Definition Language) : 데이터 정의 언어
- CREATE : 데이터 구조 생성
- ALTER : 데이터 구조 변경
- DROP : 데이터 구조 삭제
### 3. DCL (Data Control Language) : 데이터 제어 언어
- GRANT : 권한 부여
- REVOKE : 권한 철회
        
        
### 4. TCL (Transaction Control Language) : 트렌잭션 제어 언어
- COMMIT : 트랜잭션 확정
- ROLLBACK : 트랜잭션 확정
